# LearningPlaywrightNeha

Small collection of Playwright/JS learning examples and notes.

Prerequisites
- Node.js (14+)
- npm

Setup

```bash
npm install
npx playwright install
```

Run tests

```bash
npx playwright test
```

Project layout
- `Chapter_01_Basics/` - basic JS and Playwright setup examples
- `Chapter_02_Java_Basic_concepts/` - placeholder for Java notes

https://github.com/nehaarora909git/Learningplayrightneha.git
